# Food Delivery Database System

This project is a database-driven Food Delivery System designed to manage and simulate online food ordering operations.

## Features
- Manage customers, restaurants, and orders
- Store and retrieve food item details
- Order placement and tracking
- Database queries for data management

## Technologies Used
- SQL / MySQL
- Database Design (ER Model)
- Tables, Relationships, Constraints
- Basic CRUD Operations

## Database Components
- Customer Table
- Restaurant Table
- Menu Table
- Orders Table
- Order Details Table

## Description
Developed a Food Delivery Database System to understand relational database design and management. The project includes creating multiple tables with proper relationships and performing operations like inserting, updating, and retrieving data using SQL queries.

## Key Learnings
- Database schema design
- Writing efficient SQL queries
- Understanding relationships (Primary Key, Foreign Key)
- Data normalization concepts
